Project S-7 (Working Title)
Overview
2D пиксельная RPG с видом сверху, вдохновлённая классикой уровня Shining Force 2.
Проект строится вокруг перемещения по сетке, взаимодействия с объектами и модульной архитектуры (диалоги, двери, UI, инвентарь).
Основная цель — сделать чистую, расширяемую систему без излишней сложности на раннем этапе.
________________________________________
Core Systems (Implemented)
1. Movement (Tile-based)
•	Перемещение по сетке 
•	Размер клетки: 32×32 
•	Движение пошаговое (клетка → клетка) 
•	При удержании кнопки движение непрерывное 
•	Персонаж может разворачиваться на месте 
•	Перемещение блокируется: 
o	слоем непроходимых тайлов 
o	NPC 
o	закрытыми дверями 
________________________________________
2. Player Interaction System
•	Кнопка взаимодействия 
•	Взаимодействие работает только: 
o	если объект перед персонажем 
o	с учётом направления взгляда 
•	Используется зона взаимодействия перед игроком 
•	Объекты вызывают свою логику взаимодействия самостоятельно 
________________________________________
3. Dialogue System
•	Диалог запускается при взаимодействии с NPC 
•	Окно диалога: 
o	появляется при начале 
o	пролистывается кнопкой действия 
•	Во время диалога: 
o	движение отключено 
o	поворот на месте отключён 
________________________________________
4. Door System
•	Дверь — отдельный объект 
•	Имеет два состояния: 
o	закрыта 
o	открыта 
•	При взаимодействии: 
o	меняет состояние 
o	меняет внешний вид 
•	Закрытая дверь: 
o	блокирует проход 
•	Открытая дверь: 
o	позволяет пройти внутрь 
•	За дверью находится переход в другую локацию 
________________________________________
5. Interaction Architecture
Все интерактивные объекты используют единый принцип:
•	игрок инициирует взаимодействие 
•	объект сам определяет, что делать 
Это применяется к:
•	NPC 
•	дверям 
•	будущим объектам (сундуки, шкафы и т.д.) 
Игрок не знает тип объекта — только вызывает взаимодействие.
________________________________________
6. Menu System (In Progress)
•	Вызывается отдельной кнопкой 
•	Появляется снизу экрана 
•	Игра не ставится на паузу 
•	Управление персонажем блокируется 
Структура меню (крестом):
•	верх — инвентарь 
•	слева — магия 
•	справа — экипировка 
•	снизу — статус 
Навигация осуществляется через направления:
•	вверх 
•	вниз 
•	влево 
•	вправо 
Выбранный пункт подсвечивается рамкой.
